Kiểu Dữ Liệu Trong Java

Trong Java, kiểu dữ liệu đóng vai trò quan trọng trong việc xác định cách lưu trữ và xử lý dữ liệu. Bài viết này sẽ giúp bạn hiểu rõ về các kiểu dữ liệu cơ bản và cách sử dụng chúng một cách hiệu quả.

1. Kiểu dữ liệu nguyên thủy (Primitive Data Types)

Kiểu số nguyên

byte

Kích thước: 8-bit

Giá trị: -128 đến 127

byte soNguyen = 100;
short

Kích thước: 16-bit

Giá trị: -32,768 đến 32,767

short soNgan = 1000;
int

Kích thước: 32-bit

Giá trị: -231 đến 231-1

int soNguyen = 1000000;
long

Kích thước: 64-bit

Giá trị: -263 đến 263-1

long soLon = 1234567890L;

Kiểu số thực

float

Kích thước: 32-bit

Độ chính xác: 6-7 chữ số thập phân

float soThuc = 3.14f;
double

Kích thước: 64-bit

Độ chính xác: 15 chữ số thập phân

double soThucChinhXac = 3.141592653589793;

Kiểu ký tự và logic

char

Kích thước: 16-bit

Giá trị: Unicode characters

char kyTu = 'A';
boolean

Kích thước: 1-bit

Giá trị: true hoặc false

boolean dung = true;

2. Ví dụ thực tế

Quản lý thông tin sinh viên

public class SinhVien {
    private String hoTen;        // Chuỗi ký tự
    private int tuoi;           // Số nguyên
    private double diemTB;      // Số thực
    private boolean daTotNghiep; // Logic
    
    public SinhVien(String hoTen, int tuoi) {
        this.hoTen = hoTen;
        this.tuoi = tuoi;
        this.diemTB = 0.0;
        this.daTotNghiep = false;
    }
}

3. Lưu ý quan trọng

  • Luôn chọn kiểu dữ liệu phù hợp với nhu cầu sử dụng
  • Sử dụng int cho hầu hết các trường hợp số nguyên
  • Dùng double cho các phép tính số thực cần độ chính xác cao
  • Cẩn thận với việc chuyển đổi kiểu dữ liệu (type casting)
  • Kiểu boolean chỉ dùng 1 bit nhưng thường được JVM tối ưu thành 1 byte
